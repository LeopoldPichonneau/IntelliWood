## What is IntelliWood?

IntelliWood is an integrated monitoring and management system designed for wood pellet heating installations. Widely used in the Baltic region, these systems rely on a stove equipped with automatic ignition and an auger-driven feeding mechanism.

The stove draws fuel from a metal bunker with limited capacity. Maintaining a minimum fuel level is critical: if the bunker empties, the automatic ignition system keeps trying to ignite without fuel, eventually destroying the ignition stick and requiring a costly hardware replacement.

## System architecture

All subsystems transmit data over Wi-Fi to a central server (Raspberry Pi) that:

- Complements ESP32-S2 data with real-time weather variables (temperature, wind speed)
- Hosts data for the companion mobile application
- Provides an overview of storage levels, sensor readings, and maintenance alerts
